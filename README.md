# 🪒 Kratos - Sistema de Agendamento para Barbearias 💈

O **Kratos** é um sistema moderno de agendamento de horários desenvolvido com foco na experiência do cliente e na facilidade de gestão para o barbeiro. 

Este projeto nasceu de uma necessidade real: otimizar o fluxo de agendamentos da barbearia do meu marido, substituindo as marcações manuais por uma solução digital ágil e atrativa.

## 🚀 Status do Projeto
O Kratos é um **software vivo** e está em constante evolução. Novas funcionalidades e melhorias serão adicionadas continuamente à medida que avanço nos meus estudos de desenvolvimento.

## 🛠️ Próximos Passos (Planejamento)

### **Fase 1: Interface Visual (HTML/CSS)**
- [ ] Tela de agendamento para o cliente.
- [ ] Vitrine de serviços interativa com descrição e preços (Cabelo, Barba, Combos).
- [ ] Painel do barbeiro para ver os horários do dia.
- [ ] Tela de cadastro/login de clientes.

### **Fase 2: Lógica e Inteligência Local (JavaScript)**
- [ ] Sistema de agendamento funcional (salvando no navegador).
- [ ] Validação do cadastro de clientes (não deixar campos vazios).
- [ ] Botão para enviar confirmação direto para o WhatsApp do barbeiro.

### **Fase 3: Multi-Barbeiros e Banco de Dados (Back-end)**
- [ ] Criação de banco de dados para salvar os clientes e agendamentos de forma permanente.
- [ ] Sistema de "Status da Agenda" (Livre, Reservado, Confirmado, Em Atendimento, Concluído).
- [ ] Filtro na agenda por barbeiro (ex: ver a agenda do Barbeiro A ou do Barbeiro B).
